A lightweight, browser-based demonstration of [VERITAS Vault](https://github.com/VrtxOmega/veritas-vault) — the high-assurance AI knowledge engine that auto-captures, organizes, and intelligences developer work across AI coding sessions.

## What's Included

| File | Purpose |
|------|---------|
| `index.html` | Demo entry point |
| `demo.js` | Interactive demo logic |
| `demo.css` | VERITAS gold-and-obsidian styling |
